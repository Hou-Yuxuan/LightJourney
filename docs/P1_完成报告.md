## 二、产出文件一览

| 文件 | 核心内容 |
|------|---------|
| `backend/config.py` | pydantic-settings 环境变量管理，含 DB / JWT / DeepSeek 配置 |
| `backend/database.py` | SQLAlchemy 引擎(SQLite)、SessionLocal、Base、get_db() 依赖注入 |
| `backend/models/user.py` | User ORM：id, username, password_hash, created_at |
| `backend/models/trip.py` | Trip ORM：user_id FK, city, date, start_time, end_time, title, description, budget, 含 idx_trips_user_date 索引 |
| `backend/services/auth_service.py` | hash_password (bcrypt)、verify_password、create_access_token (JWT)、decode_access_token |
| `backend/routers/auth.py` | POST /api/auth/register（用户名唯一校验）、POST /api/auth/login（防用户名枚举） |
| `backend/dependencies.py` | get_current_user() JWT 鉴权依赖，返回 {user_id, username} |
| `backend/schemas.py` | RegisterRequest / LoginRequest Pydantic 模型 + 统一响应函数 |
| `backend/main.py` | FastAPI 入口、CORS(localhost:5173)、lifespan 启动建表、路由注册 |

---

## 三、关键技术决策

| 决策 | 选择 | 原因 |
|------|------|------|
| 配置管理 | pydantic-settings | 自动读取 .env，类型安全，IDE 提示友好 |
| 密码存储 | bcrypt | 不可逆哈希，行业标准 |
| Token 格式 | JWT (HS256, 24h) | 无状态鉴权，前端 localStorage 存储 |
| 路由注册 | 通过 main.py 注入 prefix | 各路由模块保持独立，避免路径硬编码 |
| 响应格式 | `{code, message, data}` | 统一前后端契约，全局拦截器按 code 分流处理 |
