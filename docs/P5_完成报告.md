## 三、功能实现详情

### 3.1 AI 规划弹窗 (AIPlanDialog.vue)

**Step 0 — 参数填写：**
- 目的地（城市名）、开始日期 + 结束日期选择器、偏好多选（美食/自然风光/人文历史）、预算上限选填
- 日期范围校验：结束日期不能早于开始日期
- 后端自动计算天数 = (结束 - 开始) + 1

**Step 1 — 预览编辑：**
- 按日期分组展示 AI 生成的行程卡片
- 点击卡片展开编辑面板，可修改日期、时间、标题、描述、预算
- 支持删除单条行程（删除后置灰不可操作）
- 内部冲突（AI 生成行程之间时段重叠）：浅黄 `#FFF8E1` 背景提示，不阻断保存
- 外部冲突（与已有行程时段重叠）：浅红 `#FFEBEE` 背景 + 冲突标签，保存时硬拦截
- 底部图例说明

**Step 2 — 确认保存：**
- 列出待保存行程清单，标记哪些会被跳过（外部冲突）
- 逐条调用 POST /api/trips 写入数据库
- 保存完成后显示结果：`已保存 X 条，Y 条因时段冲突未保存`
- 无冲突项直接保存，外部冲突项跳过

### 3.2 AI 文案弹窗 (AICopywritingDialog.vue)

- 弹窗打开时自动调用 POST /api/ai/copywriting
- 加载中显示骨架屏 `el-skeleton`
- 失败显示错误状态 + 重新生成按钮
- 成功显示可编辑文本框 + 行程信息标签
- 一键复制：`navigator.clipboard.writeText()`，成功/失败均有提示

## 四、关键技术决策

| 决策 | 选择 | 原因 |
|------|------|------|
| 天数选择 | 开始/结束日期替代数字 | 更直观，用户选具体日期而非抽象天数 |
| 编辑模式 | 点击展开编辑面板 | 行展开方式，表格整洁 + 编辑能力完整 |
| 冲突检测 | 前端双模式（内部 vs 外部） | 内部冲突仅警告，外部冲突硬拦截 |
| 批量保存 | 逐条调用 + 跳过冲突项 | 不因一条冲突阻断所有，最大化保存成功率 |
| 文案复制 | navigator.clipboard API | 原生能力，无需第三方库 |
